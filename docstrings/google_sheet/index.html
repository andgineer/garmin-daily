    function search_missed_steps_in_sheet

    search_missed_steps_in_sheet(
    fitness: Worksheet,
    rows: List[List[str]],
    columns: ColumnsMapper
) → None

    Add missed in Garmin API steps data from earlier entered in the spreadsheet.

    We need that only if we need very old data - for some reason Garmin API do not return steps more than for last three months. May be we can use some flag to get them - in the Garmin App you can see this steps but not from the API. Anyway in such situation if we already had entered steps manually in the Google Sheet we can find and use them.

    Just once I needed this mode so I keep it in the code. If you have steps from Garmin API we won't read the Google Sheet to Pandas DataFrame so this code won't take any resources.
