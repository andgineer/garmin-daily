method estimate_steps

estimate_steps() → int

Estimate steps for the activity.

Garmin do not provide steps for some activities. Actually it erroneously calculate steps during such activities like roller skiing. We have to estimate this steps to subtract them from Walking activity steps.

detect_sport(activity: Activity) → Tuple[str, bool]

Return (sport, separate)

If the activity looks like significant one we want to see it separately. For example we would like to aggregate all small bicycle trips but not the big training one.
