API Overview

Modules

  • columns_mapper: Map fields to columns using spreadsheet header row.
  • garmin_aggregations: Garmin data aggregated daily.
  • google_sheet: Export Garmin data to Google Sheet.
  • snake_to_camel: Convert names from Python (snake-name) to Java (camelName) name convention and vice-versa.
  • version

Classes

  • columns_mapper.ColumnsMapper: Map columns based on a spreadsheet header row.
  • columns_mapper.GarminCol: Columns of the Garmin daily data spreadsheet.
  • garmin_aggregations.Activity: Garmin activity.
  • garmin_aggregations.ActivityField: Activity field description.
  • garmin_aggregations.AggFunc: Activity field aggregation function.
  • garmin_aggregations.GarminDaily: Aggregate activities daily.
  • garmin_aggregations.GarminDay: Aggregate one day Garmin data.
  • google_sheet.Weekdays: Weekdays for datetime.weekday.

Functions

  • google_sheet.add_rows_from_garmin: Add activities from Garmin to the Google Sheet.
  • google_sheet.create_day_rows: Sheet rows for the day.
  • google_sheet.detect_days_to_add: Get last filled date and calculate number of days to add till today.
  • google_sheet.localized_csv_raw: Convert fields to the Google Sheet locale specific string representation.
  • google_sheet.open_google_sheet: Open Google Sheet.
  • google_sheet.search_missed_steps_in_sheet: Add missed in Garmin API steps data from earlier entered in the spreadsheet.
  • google_sheet.sheet_week_day: Weekday number for Google Sheet.
  • google_sheet.week_num: Week number for the sheet (starting from some arbitrary date).
  • snake_to_camel.camel_to_upper_snake: Convert camelCase to UPPER_SNAKE_CASE.
  • snake_to_camel.snake_to_camel: Convert snake_case_names to camelCase.
